1️⃣ Linux Fundamentals
Linux OS Basics
 What Linux is: An open-source operating system (OS) based on UNIX.
 Why used in servers: Stable, secure, lightweight, flexible, free. Most cloud servers run Linux.
 Linux distributions: Variants of Linux with different package managers and default tools:
o Ubuntu/Debian: Popular for ease of use.
o CentOS/RHEL: Enterprise servers, stable and secure.
o Differences: Package management (apt vs yum), default config files, systemctl vs service commands.
 Kernel vs User Space:
o Kernel: Core of OS, interacts directly with hardware.
o User Space: Where applications and commands run.

File Permissions & Ownership
 Permissions: r = read, w = write, x = execute
 Viewing permissions: ls -l
 -rw-r--r-- 1 user group 123 Jan 1 file.txt
o First char: file type (- regular, d directory)
o Next 3: owner permissions
o Next 3: group permissions
o Last 3: others
 Commands:
o chmod 644 file.txt → change permissions
o chown user file.txt → change owner
o chgrp group file.txt → change group
 Special permissions:
o Sticky bit: only owner can delete file in directory (chmod +t)
o setuid: file runs as owner (chmod u+s file)
o setgid: file/group inherits parent group (chmod g+s folder)

4️⃣ Package Management
